## 3. 生成精准搜索关键词

规则：
- **提取核心术语**：从问题中抽 2-3 个关键词
- **中文优先**：ima 知识库以中文为主
- **同义词扩充**：例如 "混合效应模型" → "glmmTMB / 混合效应 / mixed model"
- **不要用完整句子**：只搜关键词

## 4. 执行搜索

对每个目标知识库，调用 `ima_search_knowledge`：

```
ima_search_knowledge({ knowledge_base_id: "<kb_id>", query: "<关键词>" })
```

### 搜索结果处理
- **命中数 = 0**：换同义词再搜一次
- **命中数 > 10**：只看前 10 条（最相关的）
- **多库搜索**：每个库独立搜，结果按知识库分组呈现

## 6. 如果是统计/代码问题 → 更新手册

如果用户问的是**统计方法/R代码**相关的问题，且搜索结果中有高质量代码或方法说明：

1. 将验证过的内容补充到 `.reasonix/handbooks/stats-methods.md` 对应章节
2. 在手册的 **验证记录** 表中添加一行：
   ```
   | <日期> | <章节> | <包版本> | ima知识库搜索 | ✅ 已验证 | 来源: <知识库名> |
   ```

## 7. 注意事项

- **订阅库只读**：别人创建的知识库只能搜索，不能写入
- **自己的库**：如果发现有用内容想保存，可以写入 Moo-tivated的知识库
- **大规模库**："统计学"库有 31830 篇，搜索时关键词要尽量精准
- **不需要每次搜所有库**：只搜用户问题对应的领域知识库，省时准确

## 6. 通用搜索技巧

### 跨库搜索策略

```
问题 → 判断领域 → 路由到对应库 → 生成2-3个关键词 → 逐个搜索 → 合成结果
```

### 关键词优化原则

| ❌ 差的关键词 | ✅ 好的关键词 |
|:-------------|:-------------|
| "怎么做统计" | "R语言 t检验 代码" |
| "机器学习" | "随机森林 mlr3 调参" |
| "深度学习" | "Transformer self-attention 图解" |
| "鱼" | "长江鱼类 区系 多样性" |

### 搜不到怎么办？

1. **换同义词**："混合效应模型" → "mixed model" → "glmmTMB"
2. **拆短词**："大模型强化学习人类反馈" → "RLHF"
3. **换库**：同一个主题可能分散在不同知识库
4. **换个角度**：搜方法名 → 搜应用场景
5. **去"统计学"大库**：31830篇，90%概率能找到

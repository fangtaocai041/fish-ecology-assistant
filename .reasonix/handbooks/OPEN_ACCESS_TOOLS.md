# 开放获取工具清单 — 鱼类生态学研究

> **目的**: 汇总可用于获取学术论文、书籍、数据、代码的开放获取工具。
> ⚠️ **仅供个人学术研究使用**。使用前请自行判断法律风险。
> **更新**: 2025

---

## 🏴 论文获取工具

### Sci-Hub
- **用途**: 通过 DOI 获取付费论文 PDF
- **网址**: sci-hub.se (主站) / sci-hub.st / sci-hub.ru
- **最新域名查询**: Wikipedia 搜索 "Sci-Hub"
- **操作**: 粘贴 DOI → 输入验证码 → 下载 PDF
- **覆盖**: 85%+ 已发表论文

### Library Genesis (LibGen)
- **用途**: 论文 + 学术书籍 + 教科书
- **网址**: libgen.is / libgen.st / libgen.gs
- **操作**: 搜索标题/作者/DOI → 选择镜像 → 下载

### Anna's Archive
- **用途**: 聚合搜索 Sci-Hub + LibGen + Z-Library
- **网址**: annas-archive.org
- **优势**: 单一搜索框覆盖三大源，更稳定
- **操作**: 输入标题/DOI → 选择源 → 下载

### Z-Library
- **用途**: 书籍 + 论文
- **访问**: 需下载官方 App (桌面/移动端)
- **网址**: z-lib.org (通过 App 访问)

## ✅ 合法开放获取工具

### Unpaywall
- **用途**: 浏览器插件 — 自动检测付费论文是否有免费 OA 版本
- **网址**: unpaywall.org
- **安装**: Chrome/Firefox 扩展商店

### Open Access Button
- **用途**: 主动搜索论文的免费全文版本
- **网址**: openaccessbutton.org
- **操作**: 输入 DOI/URL → 自动查找所有 OA 副本

### EndNote Click (原 Kopernio)
- **用途**: 一键获取 PDF (通过所在机构订阅)
- **网址**: kopernio.com
- **适用**: 有机构订阅 WoS/Scopus 的用户

### Google Scholar PDF
- **用途**: 搜索结果右侧直接显示 [PDF] 链接
- **注意**: 仅对已公开的 OA 版本有效

### CORE
- **用途**: 全球最大的 OA 论文聚合器
- **网址**: core.ac.uk
- **覆盖**: 2 亿+ OA 论文

### Paperity
- **用途**: 多学科 OA 期刊聚合
- **网址**: paperity.org

## 🔍 DOI 解析技巧

如果一篇论文你只知道标题，没有 DOI：

```
1. Google Scholar 搜索标题 → 点击引用 → 复制 DOI
2. Crossref 搜索: search.crossref.org
3. 如果没有 DOI → 用 Sci-Hub 的标题搜索功能 (部分支持)
```

# 数据库与文献访问指南 — 鱼类生态学研究

> **目的**: 标注所有引用数据库的实际访问方式，区分"系统可直接访问"和"需手动操作"的资源。
> **适用领域**: 鱼类生态学、稳定同位素、几何形态测量、eDNA、保护遗传学
> **更新**: 2025

---

## 📊 访问状态标签

| 标签 | 含义 | 系统能否直接访问 |
|:----:|------|:----------------:|
| ✅ 开放 | 免费公开，无需账号 | 部分可 (通过 MCP) |
| ⚠️ 需注册 | 免费但需注册账号 | ❌ 需手动注册 |

## 四、开放获取工具

> ⚠️ **法律风险提示**: 以下工具仅供个人学术研究使用。请自行判断法律风险，遵守所在机构政策。

### 论文获取

| 工具 | 用途 | 网址 |
|------|------|------|
| **Sci-Hub** | 通过 DOI 获取付费论文 | sci-hub.se / sci-hub.st / sci-hub.ru |
| **Library Genesis** | 论文 + 书籍 | libgen.is |
| **Anna's Archive** | 聚合 Sci-Hub + LibGen + Z-Library | annas-archive.org |
| **Z-Library** | 书籍 + 论文 | 通过官方 App |
| **Unpaywall** | 浏览器插件 — 自动找 OA 版本 | unpaywall.org |
| **Open Access Button** | 搜索免费全文 | openaccessbutton.org |
| **EndNote Click** | 一键获取 PDF (机构订阅) | kopernio.com |

### Sci-Hub 操作流程

```
1. 获取论文 DOI → 2. 打开 sci-hub.se → 3. 粘贴 DOI → 4. 下载 PDF
若域名失效：查 Wikipedia "Sci-Hub" 获取最新域名
备用：Anna's Archive (更稳定聚合搜索)
```

## 八、现实世界操作清单

### 立即可做
- [ ] 注册 GBIF 账号 (gbif.org)
- [ ] 注册 BOLD Systems 账号 (boldsystems.org)
- [ ] 安装 Unpaywall 浏览器插件
- [ ] 收藏 Sci-Hub 最新域名
- [ ] 确认学校 VPN + 知网/万方可访问
- [ ] 注册中国气象数据网账号 (data.cma.cn)

### 需要申请
- [ ] 向 Cornell Bioacoustics 申请数据 (如研究涉及)
- [ ] 注册 CALIS/NSTL 文献传递

### 定期检查
- [ ] 每 6 个月更新 Sci-Hub 可用域名
- [ ] 每 6 个月检查 CRAN 包版本变化

## 🌏 中文简介

**reasonix-data** 是一个面向 **鱼类生态学 / 保护遗传学 / 水生生物学** 研究的 AI 智能体配置包。

它将 Reasonix Code 从通用编码助手转变为**专业研究团队**——集成 16 个 MCP 工具、12 个领域 AI 子智能体、5 引擎并行搜索、自动化的 5 阶段研究流水线，以及 R 统计计算环境。

### 特色
- 🐟 **鱼类学专属**：形态测量、同位素、食性分析、群落生态、遗传多样性
- 🔬 **中英双语**：同时检索中文和英文文献，覆盖本土和国际前沿
- 🧠 **13个ima知识库**：连接腾讯ima知识库，包括 31830 篇统计学库、502篇大模型库等
- 📦 **一键迁移**：克隆仓库 → 运行迁移脚本 → 5 分钟完成配置

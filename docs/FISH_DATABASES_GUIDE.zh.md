## 一、全球综合数据库

### 1. FishBase — 全球鱼类信息系统
| 项目 | 内容 |
|------|------|
| 网址 | https://fishbase.de / https://fishbase.se |
| 创立 | 1990年，Daniel Pauly & Rainer Froese（德国基尔大学/世界鱼类中心WorldFish） |
| 规模 | **35,800余种**，33万余俗名，6万余图片 |
| 数据 | 分类、形态（体长、体重、体形）、食性（营养级、食谱）、生活史（寿命、繁殖力、成熟年龄）、生长（VBGF参数）、栖息地、保护（IUCN等级） |
| 更新 | 持续更新，年度大版本 |
| 获取 | 网页浏览、rfishbase R包、HTML爬虫 |
| 前世今生 | 1990年作为国际水生生物资源管理中心（ICLARM，后更名WorldFish）项目启动。最初只覆盖200种经济鱼类。Daniel Pauly是渔业科学界最具影响力的学者之一（h指数>150）。2000年后获欧盟委员会资助大幅扩展至全球范围。2010年起由12个国家的镜像站组成的联盟维护。2023年API改版，原有的XML接口被HTML页面取代，R包改为解析HTML。**FishBase是全球鱼类学界的核心基础设施，没有之一。** |

### 2. FISHMORPH — 全球淡水鱼类形态性状数据库
| 项目 | 内容 |
|------|------|
| 网址 | https://freshwaterfish.sedoo.fr/fishmorph |
| 创立 | 2021年，Sébastien Brosse团队（法国图卢兹第三大学/Paul Sabatier大学） |
| 规模 | **9,534种**（占全球淡水鱼类73%） |
| 数据 | 10项形态比率性状：体延长比(BEl)、眼径比(REs)、口位(OGp)、体形(BLs)、胸鳍比(PFs)、尾柄比(CPt)等。从鱼类侧面照片/科学绘图测量，以无单位比率表示。 |
| 更新 | 2021年首次发布，Figshare/Kaggle持续更新 |
| 前世今生 | 2021年发表于《Global Ecology and Biogeography》。Brosse团队长期从事淡水鱼类多样性研究。数据库基于已发表的鱼类图鉴和科学插图，由多位鱼类学家独立测量后取均值。首次实现了全球尺度淡水鱼形态性状标准化。**我们已将251种长江物种的FISHMORPH数据导入系统。** |

### 3. FishTraits — 美国淡水鱼类性状数据库
| 项目 | 内容 |
|------|------|
| 网址 | https://www.usgs.gov（美国地质调查局） |
| 创立 | 2012年，Emmanuel Frimpong & Paul Angermeier（弗吉尼亚理工大学→USGS） |
| 规模 | **809种**美国淡水鱼类 |
| 数据 | 生态性状（营养级、食性类型、耐受性）+ 生活史（寿命、成熟年龄、繁殖力、产卵类型、亲体护卫） |
| 前世今生 | 最初是Frimpong的博士论文项目。当时美国没有统一的鱼类性状数据库，大量信息分散在灰色文献中。Frimpong花费数年从各州渔业报告、学术论文和专家咨询中汇编。2012年由USGS正式发布。影响了后续全球数据库的设计思路。 |

### 4. 新热带淡水鱼类生态矩阵
| 项目 | 内容 |
|------|------|
| 网址 | https://www.nature.com/articles/s41597-025-04674-w |
| 创立 | 2025年，Nature Scientific Data |
| 规模 | **6,345种**（全部新热带淡水鱼类，含南美洲、中美洲、大安的列斯群岛） |
| 数据 | **42项生态性状**：体长、4项水化学耐受（pH、硬度等）、21项物理栖息地（流速、底质、水层等）、10项食性（营养guild、摄食方式）、6项行为（洄游、集群等） |
| 前世今生 | 由新热带鱼类学家社区数十位专家协作完成，历时数年。全球最完整的单一生物地理区鱼类性状矩阵。发表于Nature子刊Scientific Data，完全开放获取。**其42项性状的分类框架是我们设计61项性状目录的重要参考。** |

## 二、保护与分类数据库

### 5. IUCN红色名录
| 项目 | 内容 |
|------|------|
| 网址 | https://iucnredlist.org |
| 创立 | 1964年（国际自然保护联盟物种生存委员会） |
| 规模 | **16万+物种**（含约3万种鱼类） |
| 数据 | 保护等级(EX灭绝/EW野外灭绝/CR极危/EN濒危/VU易危/NT近危/LC无危/DD数据缺乏)、种群趋势、威胁因子、分布范围 |
| 更新 | 每年2次 |
| 前世今生 | 始于1964年的"红皮书"传统。2000年确立v3.1版定量评估标准（种群下降率/分布面积/成熟个体数等5大准则）。2017年起开放API接口。是全球保护生物学最权威的参考标准。**我们的430种全部标注了IUCN等级。** |

### 6. Eschmeyer鱼类分类目录
| 项目 | 内容 |
|------|------|
| 网址 | https://www.calacademy.org/scientists/projects/eschmeyers-catalog-of-fishes |
| 创立 | 1998年，William Eschmeyer（加州科学院CAS） |
| 规模 | **约6万有效种**，囊括全部属级和种级分类 |
| 数据 | 接受名→异名映射、命名人、发表年份、模式产地 |
| 更新 | **每日持续更新** |
| 前世今生 | 前身是Eschmeyer 1990年出版的《鱼类属级名录》（Genera of Fishes）。1998年数字化为在线数据库。Eschmeyer本人是鱼类分类学泰斗。现由加州科学院Richard van der Laan主编，Ronald Fricke协同维护。**全球鱼类分类学的最高权威。我们的分类变更追踪直接以此为标准。** |

## 三、中国特有数据库

### 7. 中国生物多样性红色名录—脊椎动物卷
| 项目 | 内容 |
|------|------|
| 网址 | https://www.mee.gov.cn |
| 发布 | 2015年（生态环境部 + 中国科学院） |
| 规模 | **4,357种**中国脊椎动物（含内陆鱼类） |
| 数据 | 中国特有保护等级、濒危原因、种群现状、保护建议 |
| 更新 | 不定期（2015年初版→2023年更新） |
| 前世今生 | 由5个课题组分工完成：中科院动物所（哺乳类）、北师大（鸟类）、成都生物所（爬行类）、水生生物研究所（内陆鱼类）。内陆鱼类由水生所团队牵头，参考FishBase、《中国动物志》等。**这是中国最权威的物种保护评估。我们430种全部标注。** |

### 8. CNKI / 中国知网
| 项目 | 内容 |
|------|------|
| 网址 | https://cnki.net |
| 创立 | 1999年（清华大学/清华同方） |
| 规模 | 数千万篇中文学术论文、学位论文、会议论文 |
| 数据 | 中文鱼类学论文（形态描述、生态调查、性状测量数据） |
| 更新 | 每日 |
| 前世今生 | 前身是1995年《中国学术期刊（光盘版）》。1999年上线网络版。涵盖《水生生物学报》《中国水产科学》《水产学报》《湖泊科学》等核心期刊。**西方任何一个数据库（PubMed、Web of Science、Scopus）都不索引CNKI中的中文期刊——这是我们独有的数据源优势。** |

## 五、我们与国际顶尖团队的对比

| 国际标准 | 我们的做法 | 优势/差距 |
|----------|-----------|-----------|
| FishBase 35,800种 | 长江430种 + FISHMORPH 251种 | 范围小但种群级深度大 |
| FISHMORPH 10项形态 | 已导入251种比率的性状 | ✅ 已对齐国际标准 |
| Neotropical 42项性状 | 61项性状分7大类 | ✅ 覆盖更全面 |
| FishTraits 生态+生活史 | 食性全覆盖 + 栖息地全覆盖 | ✅ 已补齐 |
| IUCN 单一保护体系 | IUCN + 中国红皮书 + 国家保护 + CITES | ✅ 独家四轨制 |
| 物种级性状 | 26条水域标注的种群级性状 | ✅ 全球少有团队做 |
| 分类变化追踪 | 20条变更 + 性状冲突分析 | ✅ 创新方法 |

## 辩证综合

当 IUCN 与中国红色名录对同一物种给出不同等级时:
```
例: 圆口铜鱼 (Coreius heterodon)
  IUCN = EN (濒危)
  中国 = VU (易危)
  综合: 全球视角认为更危, 中国视角认为次之
  建议取较高保护等级(EN)作为管理基准
```
